### ⚡️ 칭호  커맨드 ⚡️️
```yml

/칭호 장착 <이름> : <이름>의 칭호를 장착합니다.

/칭호 해제 : 장착중인 칭호를 해제합니다.

/칭호 목록 : 보유중인 칭호 목록을 확인합니다.

```
<hr>

### 📑 칭호 펄미션 📑️
```yml

/칭호 장착 <이름> : customprefix.use

/칭호 해제 : customprefix.clear

/칭호 목록 : customprefix.list

```
